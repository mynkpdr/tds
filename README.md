## 🤖 LLM API Access

**AIPipe:** https://aipipe.org/

AIPipe provides access to multiple Large Language Models (LLMs) through a unified API.

### Features

* Access various AI models through a single API
* Experiment with different LLMs
* Integrate AI into assignments and projects
* Receive **$1 worth of free credits every week using IITM email**

This platform can be used for course assignments and project work.
